Funktionsweise:
In der Titelleitse kann man einen Request stellen.
Dort hat man die Möglichkeit entweder nur ein Land auszuwählen oder ein Land und eine Stadt.
Sollte man nur ein Land auswählen, werden alle Städte in dem Land abgefragt und aufgelistet.
Außerdem erscheint ganz oben ein Feld das die Durchschnittswerte für ein Ganzes Land berechnet (Alles Städte des Landes im Schnitt)
Andernfalls wird genau nach einer Stadt gesucht und diese Ausgegeben (In einer Zeile).

In den Zeilen wird folgendes Angezeigt.
Landname - Stadtname - Durchschnittstemperatur - Durchschnittsfeuchtigkeit - Durchschnittswindgeschwindigkeit - Evtl. Icon
